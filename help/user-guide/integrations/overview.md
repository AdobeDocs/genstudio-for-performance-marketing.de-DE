# Adobe GenStudio for Performance Marketing-Integrationen

GenStudio for Performance Marketing bietet Integrationen mit anderen leistungsstarken Designprodukten, die Marketing-Experten, die auf diese Produkte angewiesen sind, ein nahtloses Erlebnis bieten. Hier finden Sie Informationen zu den verfügbaren Integrationen und deren Verwendung.

Erkunden Sie die Plug-ins, die GenStudio for Performance Marketing zusammen mit anderen Adobe-Produkten und darüber hinaus bietet, um Ihre Design-Workflows zu optimieren.

## Figma-Plug-in

Das GenStudio for Performance Marketing Figma-Plug-in fügt dem Figma-Programm ein neues Bedienfeld hinzu, mit dem Benutzende GenStudio for Performance Marketing-Aktionen ausführen können, z. B. das Generieren von markeninternen Inhalten.

Erfahren Sie mehr über das Figma-Plugin auf der [Figma-Plugin](figma-plugin.md) Seite.

## Photoshop-Plug-in

Das GenStudio for Performance Marketing Photoshop-Plug-in fügt Adobe Photoshop ein neues Bedienfeld hinzu, mit dem Benutzende GenStudio for Performance Marketing-Aktionen ausführen können, z. B. das Generieren von markeninternen Inhalten.

Weitere Informationen zum Photoshop-Plug-in finden Sie auf der Seite [Photoshop](photoshop-plugin.md)Plug-in.
