```markdown
### Example feature

You can now do X by opening [!DNL Create], going to the template gallery, selecting an approved experience, clicking **[!UICONTROL Translate]** in the actions menu, choosing one or more languages from the list, waiting for the canvas to load, and then editing the translated text before you request approval.
```
